# Library Management System

## Features
- Admin and User login
- Book Add, Issue, Return
- Fine Calculation (₹10/day)
- Membership Management
- User Management
- Book Search (Report Module)

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS

## How to Run
1. Create virtual environment:
   python -m venv venv

2. Activate:
   venv\Scripts\activate

3. Install Flask:
   pip install flask

4. Run:
   python app.py

5. Open:
   http://127.0.0.1:5000

## Assumptions
- Dummy login used
- Fine = ₹10 per day
- Basic UI used
